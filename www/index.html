Ateliers Python

Antoine Mazières (Centre Marc Bloch) et Julie Pierson (Passages)

Vous trouverez sur cette page les ressources pour suivre les ateliers python de l'école thématique CNRS Explo-SHS :

  • Atelier «Analyse de données», Mercredi 14 Octobre 2020, 9h30-12h15
  • Atelier «Récolte de données», Jeudi 15 octobre 2020, 9h30-12h15

Vous avez tous des niveaux de connaissance variés en terme de programmation, d'analyse de données et de technologies Web. Pour cela nous avons essayé d'organiser ces enseignements de manière à ce que chacun puisse, nous l'espérons, y trouver un intérêt. Chaque atelier propose une introduction simple et détaillée, puis vous présente des cas d'utilisations plus avancés et moins détaillé qu'il vous appartient d'explorer en trouvant des applications proches de vos besoins en terme de recherche et d'ingénierie. Dans tous les cas, n'hésitez pas à solliciter l'aide de Julie (en présentiel) et Antoine (en distanciel).

Pour suivre ces ateliers, vous devez installer Anaconda Python. Cela dit, si vous avez déjà vos propres habitudes avec Python, n’hésitez pas à les garder pour être plus à l’aise. Nous utiliserons une interface de programmation interactive, les notebooks, afin que vous puissiez éditer le materiel et tester les codes au fur et à mesure.

Si vous n'avez jamais utilisé Python et/ou que vous avez peu ou pas d'expérience en terme de programmation, nous vous invitons à commencer par le notebook « Initiation à Python et la programmation ». Si vous souhaitez mettre à profit ces 2h45 pour apprendre Python ou progresser en programmation plutôt que de vous plonger dans les usages et bibliothèques spécifiques proposés lors de ces ateliers, n'hésitez pas ! Un lien à la fin de ce premier notebook vous renvoie vers une plateforme avec de nombreux exercices pour ce faire.

Pour pouvoir accéder aux contenus:

  • Décompressez le fichier explo-shs-python.zip
  • Ouvrez un Terminal :
    • Sur Windows : Cherchez «Anaconda Prompt» dans le menu Windows
    • Sur Mac : Applications > Utilitaires > Terminal
  • Depuis le terminal, déplacez-vous dans le dossier où se trouve le dossier téléchargé : 
    cd Downloads/explo-shs-python/
  • Depuis le terminal, lancez l'application Jupyter Notebook : 
    jupyter notebook

Amusez-vous bien ! Les solutions aux exercices seront disponibles sur ce site à la fin de chaque atelier.
